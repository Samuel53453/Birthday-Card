# 🎂 Birthday Card

A simple birthday card webpage built with HTML and CSS.
The page features styled text, images, and GIFs to celebrate someone's birthday in a fun and visual way.

This project was created as part of the **Scrimba Fullstack Developer Path**, focusing on practicing CSS styling and understanding layout positioning.

## ✨ Features

* **Hover Interaction** – The birthday image changes into a GIF when hovered.
* **Styled Birthday Message** – Decorative typography and layout for a celebratory feel.
* **Flexbox Layout** – Elements are structured using Flexbox for better alignment.
* **Visual Styling** – Custom colors, gradients, and images to enhance the design.

## 🧠 What I Learned

While building this project, I practiced several important CSS concepts:

* **align-items** – Learned how to vertically align elements inside a Flexbox container.
* **flex-direction** – Used `column` and `row` to control how elements are arranged on the page.
* **:hover pseudo-class** – Applied hover effects to change an image into a GIF when users interact with it.
* **linear-gradient** – Used gradients to create more visually appealing backgrounds.
* **CSS grouping selectors** – Applied the same styles to multiple elements using comma-separated selectors.

## 🧩 Challenges

Some challenges I faced while building this project included:

* Centering elements properly using CSS
* Remembering to include correct `alt` attributes for images
* Positioning and styling elements to achieve the desired layout

## ⚙️ The Process

I started by creating the HTML structure for the first `div` and text section.
Once the structure worked, I reused and copied it for the remaining sections.

I used **CSS utility classes** and applied **Flexbox with `flex-direction: column`** to structure the text and layout consistently across the page.
